1. 문제 발견 및 기회 창출 
    1) pain point 
        - 모든 것을 고객의 페인 포인트에서 시작해야 함 
    2) 고객 페인 포인트 수집
        - 고객센터
        - 앱 리뷰
        - SNS 
        - 서비스 오류 로그 
        - 사용자조사  
    3) 페인 포인트 분석 (예)
        - 제품 및 속성에 대한 분류 체계 마련 
            - 카테고리 : B2C 개인고객, 뱅킹        
            - 대분류   : 지메일,       내역         
            - 중분류   : 메일보내기,    거래내역
            - 소분류   : 첨부파일,      기간선택 

2. 페인포인트를 바탕으로 제품 주요기능 정의 (예)
    1) 과일배달
        - 메뉴선택
        - 주문하기
        - 결제하기 
        - 정기배송
            - 요일/시간 설정 가능
            - 원하는 과일 선택 가능 

3. STP
    1) Segmentation : 시장세분화
        - 고객을 아주 작은 집단으로 나누는 것
    2) Targeting : 표적시장
        - 구분된 소비자 층중에서 우리 서비스를 선택할 
          가능성이 높은 핵심 소비자층을 표적시장으로 선별 
          - 예) 여행상품 구독서비스를 제공
                - 매년 해외여행을 하는 사람,
                  구독서비스에 거부감이 없는 사람,
                  월 10만원을 여행 상품 검색에 투자할수 있는 소비자 
    3) Positioning : 포지셔닝 
        - 제품을 인지시키는 Identity, Image를 설정하는 것 
            - 예) 자이리톨 껌 : 충지예방
                  에이스침대 : 과학적인 침대, 편안한 침대
                  카카오톡 : 국민 메신저
    4) 어떤 사람들 중 누구에게 어떤 가치를 제공할 것인지를 설정하는 것

4. SWOT 분석 
    1) 환경 분석하는데 자주 사용하는 도구 
        - Strengths (강점)
        - Weakness (약점)
        - Opportunies (기회)
        - Treats (위협)    

5. 리서치 
    1) 리서치 하는 이유
        - 새로운 사업/서비스 출시에 앞서 당위성을 찾기 위함  
        - 변화하는 시장환경에 대응 할 수 있는 서비스를 출시하기 위함 

    2) 리서치 방법
        - 기사 
        - 검색 (구글, 네이버 등)               
        - 증권사 리포트 
        - 뉴스레터 구독

    3) UX 리서치 
        - 제품에 대한 시장의 반응 및 선호도를 조사하는 것
        - 더 나아가 사용자들의 행동과 태도를 관찰하고 
          문제 해결방법의 근거를 찾아서 실질적인 해법을 찾는 것을 의미         

6. 사용자 & 서비스 분석 - A/B 테스트  
    1) 사례
        - 오마바 전 미국 대통령        

7. UI / UX 
    1) UI (User Interface)         
        - 사용자가 제품/서비스를 사용할 때 마주하게 되는 시각적인 부분 
        - 폰트, 컬러, 레이아웃, 로고 등의 요소를 말함 

    2) UX (User Experience)        
        - 사용자가 제품/서비스를 경험할 때 느끼는 모든 순간의 만족을 의미 
        - 사용자들의 경험 데이터를 분석하여 더 편하고 효율적인 방향으로
          사용자 중심적으로 설계된 UI (디자인)
